# Day17_EmployeeSalaryAnalysis-
This program performs a complete salary analysis for up to 10 employees and includes student identification details (Name, Registration Number, and Section) to make the submission unique.

 Main Features:

Student Information Input

The user (student) enters:

Name

Registration Number

Section

Employee Salary Input

The HR (user) enters:

Number of employees (maximum 10)

Salary for each employee

Menu-Driven Salary Analysis

The program repeatedly displays a menu until the user chooses Exit.

Menu Options:

Display All Salaries
→ Lists all employee salaries.

Display Total and Average Salary
→ Calculates total and average salary.

Display Highest and Lowest Salary
→ Finds and displays the highest and lowest salary.

Count Employees Above and Below Average
→ Counts how many employees earn above or below the average salary.

Display Sorted Salaries (Ascending Order)
→ Sorts salaries using Bubble Sort and displays them.

Exit
→ Ends the program.

Personalized Output

Every result section displays:
Submitted by: <Student Name> | Reg No: <RegNo> | Section: <Section>
along with
Challenge Day 17 – Code2Grow Program

Logic Used

1D Array → to store employee salaries.

Loops and conditionals only → no functions, no pointers, no structures.

Bubble Sort Algorithm → for sorting salaries in ascending order.

Do-While Loop → to repeat the menu until the user exits.

Average Calculation → (sum of salaries) / (number of employees).

Comparisons → to find highest, lowest, and above/below average salaries.
